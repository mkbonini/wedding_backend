Password Protected

The folder is password protected. To add or view photos, you will need to add the password, which is: deercreek

 

To Add Pictures

1. Go to https://dropevent.com/mikemiwha/upload

2. Add the password 

3. Put in your email address as prompted

4. Upload your photos

 

To View Pictures

1. Go to https://dropevent.com/mikemiwha

2. Add the password

 

To Download an Individual Picture

1. Go to https://dropevent.com/mikemiwha

2. Add the password

3. Select a photo

4. Click download at the top of the page 

 

Note

If you are on your phone, you will have to go to your downloads, open the file and then save the image. That will add it to your photo library.

 

To Download Bulk Images

1. Go to https://dropevent.com/mikemiwha/zip

2. Add the password

3. Select the folders you wish to download, and then download the zip folder.

 

Let us know if you have any questions about how to use this! We noticed that the images look compressed when viewing on the site, but if you actually download the photos they are in their original quality.
